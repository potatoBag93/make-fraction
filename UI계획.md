## 🖥️ 분수 감각 향상 웹앱 UI 계획서

### 1. 🧭 전체 화면 구성 개요

```
┌────────────────────────────────────────────┐
│ [문제 지시 문장] "1과 3/4을 채워보세요!"   (중간 상단 고정) │
├────────────────────────────────────────────┤
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒ │
│ ◀ 도형 영역 (화면 좌측 70%)               ▶ 결과 영역 (우측 30%) ▶ │
│ ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒ │
├────────────────────────────────────────────┤
│ [초기화] [건너뛰기] [다음]    타이머: ⏱ 08초 │  ← 하단 고정 버튼 바
└────────────────────────────────────────────┘
```

### 2. 🧩 주요 구성 요소

#### 📍 문제 지시 영역

* 화면 **중간 상단**에 크고 명확하게 텍스트 표시
* 예: "1과 3/4을 채워보세요!"

#### 📍 도형 인터랙션 영역 (왼쪽 70%)

* 도형은 원, 네모, 물컵 등 랜덤
* 클릭 또는 드래그로 채우기 (정확한 비례 반응)
* **실시간 수치는 미표시**, 직관적 감각 훈련 유도
* 도형 내부에 겹쳐 정답 색상이 시각화됨

#### 📍 결과 피드백 영역 (오른쪽 30%)

* 정답 분수 표시: 예) "정답: 1과 3/4"
* 점수: 예) "정확도 85점"
* 문장 피드백: "1과 3/4은 1과, 4분의 3입니다."

#### 📍 하단 조작 버튼

* **초기화 버튼**: 현재 도형 초기화
* **건너뛰기 버튼**: 사용 횟수 제한 있음
* **다음 버튼**: 항상 표시되며 다음 문제로 이동 가능
* **타이머**: 숫자 표시 (예: 08초)

### 3. 🎛️ 진행 상태 및 시각 요소

* 현재 문제 번호 표시: "5 / 10"
* 모든 요소는 **한 화면 내 표시**, 스크롤 없음
* 버튼은 **100px 이상 크기**, 반응형 적용
* 드래그 진입/해제 시 안내선 표시 예정

### 4. 🎓 교사용 모드 및 수업 지원 UI 요소

* 문제 건너뛰기 횟수 설정 가능
* 정답 숨기기 기능 → 단체 풀이 시 사용
* 사용법 보기 버튼 → 튜토리얼 모드 진입
* 결과 화면: 모든 문제/입력값/정답/점수 확인 가능

### 5. 📱 디바이스 대응

* 전자칠판(터치 기반), 태블릿, 데스크탑 최적화
* 가로 모드 고정, 전체 요소 확대 표시 고려
